Privacy

This is the publicly hosted version of the Arithmetica bot.
It uses the exact same source code hosted on GitHub
By using this bot and/or by inviting it to your server, you agree to the following data being collected and stored:

  • Server IDs
  • Counting Channel IDs
  • User IDs
  • Message IDs & Content in the Counting Channel
  • Content of slash commands directed at the bot

Data the bot WILL NEVER collect:

  • Messages outside of the counting channel
  • Messages from DMs
  • User presence data
  • Server presence data
  • User names & Server names
  • User connections / bio / status

The data is hosted on a non-public secure server and is only used for the purpose of the bot.
